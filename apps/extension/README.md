# Stashly Chrome Extension

This extension reuses the existing backend API to provide:
- Sign in and sign up
- Save link
- Save current tab
- Search semantic recall
- View all saved links
- Delete saved links

## Load in Chrome
1. Open Chrome and go to chrome://extensions
2. Enable Developer mode
3. Click Load unpacked
4. Select the folder apps/extension

## Backend Requirement
Keep backend API running at http://localhost:3000.
If your API runs elsewhere, update API_BASE_URL in popup.js.
